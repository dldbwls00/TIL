	2. 잘못된 user id인 경우 - 올바르지 않다는 메시지 출력
2. password 입력
	1. 올바른 password인 경우 - 로그인 성공
	2. 잘못된 password인 경우 - 올바르지 않다는 메시지 출력
3. 로그인 버튼 클릭

			1. 비밀번호를 잘못 입력한 경우
				1. 지불 시스템의 오류 처리 절차에 따름
					1. 정상 승인된 경우 - 1.2.2로 이동
					2. 승인이 이루어지지 않은 경우 - 거래 중단
			2. 비밀번호를 제대로 입력한 경우
